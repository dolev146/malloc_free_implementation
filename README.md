# Custom Malloc and Free
A simple malloc and free implementation that uses a link list to manage memory.

## Running the test program
1. Compile using make.
   ```console
   $ make
   ```
2. Run test.
   ```console
   $ ./test
   ```
3. Run test.
   ```console
   $ make valgrind
   ```
